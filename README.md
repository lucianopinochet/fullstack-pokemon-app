# Pokemon Api
## for usage:

### CLIENT:
```
  -cd ./client
  -npm i
  -npm run dev
```
### SERVER:
```
  -cd ./server
  -npm i
  -add .env file with:
    -MONGO_URL='{Cluster connection url to mongodb}'
    -JWT_KEYWORD='{Keyword of preference !!!Try to don't change through the course of development}'
    -PORT={Number of preference, try to use a not common port}
  -npm run dev
```
